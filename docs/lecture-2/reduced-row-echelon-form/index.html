Reduced Row Echelon Form

Definitionâ€‹

Supposed an augmented matrix is in row-echelon form. It is in reduced row-echelon form if :

  1. The leading entry of every non-zero row is 1:
  • Equivalently, every pivot point is 1
  1. In each pivot column, except the pivot point, all other entries are 0.
