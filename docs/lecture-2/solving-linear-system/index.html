Solving Linear Systems

Supposed that the augmented matrix of a linear system is in (reduced) row-echelon form

  • How do we find a solution to the linear system?

Algorithm to solving a linear system​

Suppose that the augmented matrix corresponding to a linear system is in row-echelon form.

  1. Set the variables corresponding to non-pivot columns to be arbitrary parameters
  2. Solve the variables corresponding to pivot columns by back substitution (from last equation to first).

Example 1​

Zero system​

  • Augmented matrix (00000000)\begin{pmatrix}\begin{array}{ccc|c} 0 & 0 & 0 & 0\\ 0 & 0 & 0 & 0 \end{array}\end{pmatrix}(00​00​00​00​​)
  • Linear system {0x1+0x2+0x3=00x1+0x2+0x3=0\begin{cases} 0x_1 + 0x_2 + 0x_3 = 0\\ 0x_1 + 0x_2 + 0x_3 = 0 \end{cases}{0x1​+0x2​+0x3​=00x1​+0x2​+0x3​=0​
  • This is a zero system in three variables. It has infinitely many solutions
  • x1=r,x2=s,x3=tx_1 = r, x_2 = s, x_3 = tx1​=r,x2​=s,x3​=t and r,s,tr,s,tr,s,t are arbitrary parameters

Linear System​

  • Augmented matrix (314021001)\begin{pmatrix}\begin{array}{cc|c} 3 & 1 & 4\\ 0 & 2 & 1\\ 0 & 0 & 1 \end{array}\end{pmatrix}⎝⎛​300​120​411​​⎠⎞​
  • Linear system {3x1+1x2=40x1+2x2=10x1+0x2=1\begin{cases} 3x_1 + 1x_2 = 4\\ 0x_1 + 2x_2 = 1 0x_1 + 0x_2 = 1 \end{cases}{3x1​+1x2​=40x1​+2x2​=10x1​+0x2​=1​
  • We can see directly from the row-echelon form that the system is inconsistent.
  • Last entry is pivot!

Example 2​

  • Augmented matrix (1−103−20012500000)\begin{pmatrix}\begin{array}{cccc|c} 1 & -1 & 0 & 3 & -2\\ 0 & 0 & 1 & 2 & 5\\ 0 & 0 & 0 & 0 & 0 \end{array}\end{pmatrix}⎝⎛​100​−100​010​320​−250​​⎠⎞​

  • Linear system {1x1−1x2+0x3+3x4=−20x1+0x2+1x3+2x4=50x1+0x2+0x3+0x4=0\begin{cases} 1x_1 - 1x_2 + 0x_3 + 3x_4 = -2\\ 0x_1 + 0x_2 + 1x_3 + 2x_4 = 5\\ 0x_1 + 0x_2 + 0x_3 + 0x_4 = 0 \end{cases}⎩⎪⎪⎨⎪⎪⎧​1x1​−1x2​+0x3​+3x4​=−20x1​+0x2​+1x3​+2x4​=50x1​+0x2​+0x3​+0x4​=0​

  • Linear system(simplified) {x1− x2+ 3x4=−2x3+ 2x4=5\begin{cases}\begin{array}{ccccc} x_1 & -\:x_2 & & + \:3x_4 & =-2\\ & & x_3 & + \:2x_4 & = 5 \end{array}\end{cases}{x1​​−x2​​x3​​+3x4​+2x4​​=−2=5​​

  • We cannot solve for x4x_4x4​, we should set the value of x4x_4x4​ and solve for others

  1. i.e. Let x4=tx_4 = tx4​=t and substitute into the second equation
    • x3+2t=5⇒x3=5−2tx_3 + 2t = 5 \rArr x_3 = 5 - 2tx3​+2t=5⇒x3​=5−2t
  2. Substitute x4=tx_4 = tx4​=t into the first equation
    • x1−x2+3t=−2x_1 - x_2 + 3t = -2x1​−x2​+3t=−2
    • Let x2=sx_2 = sx2​=s. Then x1=−2+s−3tx_1 = -2 + s - 3tx1​=−2+s−3t
  • Infinitely many solutions (sss & ttt are arbitrary parameters)
  • x1=−2+s−3t,x2=s,x3=5−2t,x4=tx_1 = -2 + s - 3t, \\x_2 = s, \\x_3 = 5-2t, \\x_4 = tx1​=−2+s−3t,x2​=s,x3​=5−2t,x4​=t

Example 3​

  • Augmented matrix (0221−22001113000024)\begin{pmatrix}\begin{array}{ccccc|c} 0 & 2 & 2 & 1 & -2 & 2\\ 0 & 0 & 1 & 1 & 1 & 3\\ 0 & 0 & 0 & 0 & 2 & 4 \end{array}\end{pmatrix}⎝⎛​000​200​210​110​−212​234​​⎠⎞​
  • Linear system(simplified) {2x2+ 2x3+   x4− 2x5=2      x3+ 2x4+   x5=3    2x5=4\begin{cases}\begin{array}{ccccc} 2x_2 & +\:2x_3 & + \:\:\:x_4 & - \:2x_5 & =2\\ & \:\:\:\:\:\:x_3 & + \:2x_4 & +\:\:\:x_5 & = 3\\ & & & \:\:\:\:2x_5 & = 4 \end{array}\end{cases}⎩⎪⎨⎪⎧​2x2​​+2x3​x3​​+x4​+2x4​​−2x5​+x5​2x5​​=2=3=4​​
  1. By the third equation, 2x5=4⇒x5=22x_5 = 4 \rArr x_5 = 22x5​=4⇒x5​=2
  2. Substitute x5=2x_5 = 2x5​=2 into the second equation:
    • x3+x4+2=3x_3 + x_4 +2 =3x3​+x4​+2=3
    • x3+x4=1x_3 + x_4 = 1x3​+x4​=1
    • Let x4=tx_4 = tx4​=t. Then x3=1−tx_3 = 1 - tx3​=1−t
  3. Substitute x5=2,x3=1−t,x4=tx_5 = 2, x_3 = 1 - t, x_4 = tx5​=2,x3​=1−t,x4​=t into the first equation:
    • 2x2+2(1−t)+t−2∗2=22x_2 + 2(1-t) + t - 2 * 2 = 22x2​+2(1−t)+t−2∗2=2
    • So x2=2+12tx_2 = 2 + \frac{1}{2}tx2​=2+21​t
  4. x1=sx2=2+12tx3=1−tx4=tx5=2x_1 = s\\x_2 = 2+\frac{1}{2}t\\x_3=1-t\\x_4 =t\\x_5 = 2\\x1​=sx2​=2+21​tx3​=1−tx4​=tx5​=2where sss & ttt are arbitrary parameters.
