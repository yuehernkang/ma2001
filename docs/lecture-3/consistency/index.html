Consistency

Suppose AAA is the augmented matrix of a linear system, and RRR is a row-echelon form of AAA.

  • When the system has no solution? (i.e. is inconsistent?)
  • When the system has exactly one solution?
  • When the system has infinitely many solutions?

Recall the procedure of finding solution:

  1. Set the variables corresponding to non-pivot columns as arbitrary parameters.
  2. Solve variables corresponding to pivot columns.

The procedure is valid as long as:

  • Every row of RRR corresponds to a consistent equation.
  • i.e. no row corresponds to an inconsistent equation:
    • 0x1+0x2+⋯+0xn=⊗0x_1 + 0x_2 + \dots + 0x_n = \otimes0x1​+0x2​+⋯+0xn​=⊗

When the system has no solution? (i.e. is inconsistent?)​

  • There is a row in RRR with the form

    • (00…0∣⊗)\begin{pmatrix} 0 & 0 & \dots 0 \vert \otimes \end{pmatrix}(0​0​…0∣⊗​), where ⊗\otimes⊗ is nonzero. Or equivalently, the last column is a pivot column.

      Note: Such a row must be the last nonzero row oF RRR

      Therefore, we just use row-echelon form to check the consistency of the linear system.

      (32340005)\begin{pmatrix} 3 & 2 & 3 & 4\\ 0 & 0 & 0 & 5 \end{pmatrix}(30​20​30​45​)

When the system has exactly one solution?​

  • For consistency, the last column is non-pivot. We also need

    • No variables corresponding to non-pivot columns

  • The last column is a non-pivot column, and

  • All other columns are pivot columns.

When the system has infinitely many solutions?​

  • The last column is a non-pivot column, and
  • Some other columns are non-pivot columns

Note: The number of arbitrary parameters is the same as the number of non-pivot columns (except the last column).

Example:

(1234 0−1010012)\begin{pmatrix}\begin{array}{ccc|c} 1 & 2 & 3 & 4\\\ 0 & -1 & 0 & 1\\ 0 & 0 & 1 & 2\\ \end{array}\end{pmatrix}⎝⎛​1 00​2−10​301​412​​⎠⎞​
