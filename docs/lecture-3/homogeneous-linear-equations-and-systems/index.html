Homogeneous Linear Equations & Systems

Definition​

A linear equation in the variables x1,x2,…,xnx_1, x_2, \dots, x_nx1​,x2​,…,xn​ is called homogeneous if it is of the form

  • a1x1+a2x2+⋯+anxn=0a_{1}x_{1} + a_{2}x_{2} + \dots + a_{n}x_{n} = 0a1​x1​+a2​x2​+⋯+an​xn​=0

A linear equation x1,x2,…,xnx_1,x_2,\dots,x_nx1​,x2​,…,xn​ is homogeneous if

  • ⇔x1=0,x2=0,…,xn=0\hArr x_1 = 0, x_2 = 0, \dots, x_n = 0⇔x1​=0,x2​=0,…,xn​=0 is a solution

  • A linear equation is homogeneous if everything =0=0=0 is a solution

A linear system is homogeneous if every linear equation of the system is homogeneous.

  • {a11x+a12x2+⋯+a1nxn=0a21x+a22x2+⋯+a1nxn=0⋮am1x1+am2x2+⋯+a1nxn=0\begin{cases} a_{11}x+a_{12}x_{2}+\dots+a_{1n}x_{n}= 0\\ a_{21}x+a_{22}x_{2}+\dots+a_{1n}x_{n} = 0\\ \vdots\\ a_{m1}x_1+a_{m2}x_{2}+\dots+a_{1n}x_{n} = 0\\ \end{cases}⎩⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎧​a11​x+a12​x2​+⋯+a1n​xn​=0a21​x+a22​x2​+⋯+a1n​xn​=0⋮am1​x1​+am2​x2​+⋯+a1n​xn​=0​

A linear system x1,x2,…,xnx_1,x_2,\dots,x_nx1​,x2​,…,xn​ is homogeneous

  • ⇔x1=0,x2=0,…,xn=0\hArr x_1 = 0, x_2 = 0, \dots, x_n = 0⇔x1​=0,x2​=0,…,xn​=0 is a solution

  • A linear system is homogeneous if everything =0=0=0 is a solution

  • A homogeneous linear system is always consistent

  • This is the trivial solution of a homogeneous linear system.

  • Other solutions are called non-trivial solutions
