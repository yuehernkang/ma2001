For BABABA to contain ImI_mIm​, for the first mmm columns, AAA has to contain B−1B^{-1}B−1

Therefore, AAA's first mmm columns will always be an identity matrix.

So AAA always have a unique solution.

(By Theorem of Invertible Matrices, Chapter 2, Slide 89)

Suppose m≤nm\le nm≤n

For the first mmm columns, the RREF of AAA is III and the solution will be consistent and has infinitely many solutions.

Since the linear system is consistent for all possible cases, the linear system always has a solution.
