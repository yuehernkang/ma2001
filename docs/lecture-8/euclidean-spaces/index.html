Euclidean nnn-space​

The Euclidean nnn-space (or simply nnn-space) is the set of all nnn-vectors of real numbers.

Rn={(v1,v2,…,vn)∣v1,v2,…,vn∈R}\reals^{n} = \{(v_1, v_2, \dots, v_n) | v_1, v_2, \dots, v_n \isin \reals\}Rn={(v1​,v2​,…,vn​)∣v1​,v2​,…,vn​∈R}

v∈Rnv \isin \reals^{n}v∈Rn if and only if vvv is of the form

  • v=(v1,v2,…,vn)v = (v_1, v_2, \dots, v_n)v=(v1​,v2​,…,vn​) for real numbers v1,v2,…,vnv_1, v_2,\dots, v_nv1​,v2​,…,vn​

In particular,

  • If n=1n=1n=1, then R=R1\reals = \reals^1R=R1 is the real line
  • If n=2n=2n=2, then R2\reals^2R2 is the xyxyxy-plane
  • If n=3n=3n=3, then R3\reals^3R3 is the xycxycxyc-space
