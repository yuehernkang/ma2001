Determinant of 3x3 matrix

Let AAA be a square matrix. It is expected that

  • det(III) = 111
  • AAA is invertible ⇔det(A)≠0\hArr det(A) \not ={0}⇔det(A)=0
    • Equivalently, AAA is singular ⇔det(A)=0\hArr det(A) = 0⇔det(A)=0

We hope the determinant can determine the invertability of AAA

When :
AAA is invertible ⇔det(A)≠0\hArr det(A) \not ={0}\\⇔det(A)=0 AAA is singular ⇔det(A)=0\hArr det(A) = 0⇔det(A)=0

Properties of Determinants of 3x3 matrices​

  • A→cRiB⇔det(B)=cdet(A)A \xrightarrow{\mathbf{c}R_i} B \hArr det(B) = \mathbf{c}det(A)AcRi​​B⇔det(B)=cdet(A)
  • A→Ri↔RjB⇒det(B)=−det(A)A \xrightarrow{R_i \harr R_j}B \rArr det(B) = -det(A)ARi​↔Rj​​B⇒det(B)=−det(A)
  • A→Ri+cRkB⇒det(B)=det(A)A \xrightarrow{R_i + \mathbf{c}R_k}B \rArr det(B) = det(A)ARi​+cRk​​B⇒det(B)=det(A)

Working backwards​

Suppose AAA is invertible. Then there exist elementary row operations

  • A→ero1A1→ero2A2⇒⋯→Ak−1→erokAk=IA \xrightarrow{ero1} A_1 \xrightarrow{ero2} A_2 \xRightarrow{} \dots \xrightarrow{} A_{k-1} \xrightarrow{erok} A_k = IAero1​A1​ero2​A2​​⋯​Ak−1​erok​Ak​=I
  • Then det(A) can be evaluated backwards.
