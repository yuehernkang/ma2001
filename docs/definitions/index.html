Invertible Matrix​

Main Theorem for Invertible Matrices​

  1. AAA is an invertible matrix
  2. Linear System Ax=bAx = bAx=b has a unique solution
  3. Linear System Ax=0Ax = 0Ax=0 has only the trivial solution
  4. The reduced row-echelon form of AAA is III
  5. AAA is the product of elementary matrices

If AAA is singular, then its inverse, A−1A^{-1}A−1 is undefined

Cancellation Law​

Let AAA be an invertible matrix

  • AB1=AB2⇒B1=B2AB_1 = AB_2 \rArr B_1 = B_2AB1​=AB2​⇒B1​=B2​
  • C1A=C2A⇒C1=C2C_{1}A = C_{2}A \rArr C_1 = C_2C1​A=C2​A⇒C1​=C2​
