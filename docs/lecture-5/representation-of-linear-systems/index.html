Problem​

Suppose that a linear system has more than one solutions. Then it has infinitely many solutions

Proof​

Let the system be prepresented by Ax=bAx = bAx=b

  • Suppose that it has two solutions u1≠u2u_1 \not ={u_2}u1​=u2​
    • Au1=bAu_1 = bAu1​=b and Au2=bAu_2 = bAu2​=b
    • Solution set to a linear system must be geometrically flat. You can draw a line passing through u1u_1u1​ and u2u_2u2​.
    • 0=b−b=Au1−Au2=A(u1−u2)0=b-b=Au_1-Au_2=A(u_1-u_2)0=b−b=Au1​−Au2​=A(u1​−u2​)
  • Then for any real number, ttt
    • 0=t0=tA(u1−u2)=A(t(u1−u2))0 = t0 = tA(u_1-u_2) = A(t(u_1-u_2))0=t0=tA(u1​−u2​)=A(t(u1​−u2​)) is always 0
    • b=b+0=Au2+A(t(u1−u2))b = b + 0 = Au_2 + A(t(u_1-u_2))b=b+0=Au2​+A(t(u1​−u2​))
    • =A(u2+t(u1−u2))=A(u_2 + t(u_1-u_2))=A(u2​+t(u1​−u2​)) is always a solution for any value of ttt.

So u2=t(u1−u2)u_2 = t(u_1 - u_2)u2​=t(u1​−u2​) is a solution to Ax=bAx = bAx=b for every real number ttt.

Therefore, the system has infinitely many solutions.
